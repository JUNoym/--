git commandを理解する

・git clone 
  githubからリポジトリを持ってくるコマンド

・git branch
  git branchコマンドを使うと、存在するブランチを表示できる

・git checkout
  git checkoutコマンドを使いブランチ名を指定すると、そのブランチに移動できる
  
・git log
  git logコマンドを使うと、コミットの詳細を表示できる

・git status
  ワーキングディレクトリやインデックスの状態を確認するためのコマンド

・git add
  コミット対象ファイルをインデックスに登録するためのコマンド

・git commit
  インデックスに登録されたファイルをコミットするためのコマンド

・git diff
  コミット前のファイルとコミット後のファイルの差分を表示するためのコマンド

・git push
  ローカルリポジトリの内容をリモートリポジトリに反映させるためのコマンド

・git pull
  「git fetch」と「git merge」の２つをまとめて実行するコマンド

・git fetch
  リモートリポジトリの情報をローカルリポジトリに反映するためのコマンド

・git merge
  特定のブランチやリビジョンを、現在のブランチに取り込むためのコマンド
