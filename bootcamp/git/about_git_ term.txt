GA Tryoutを進めるために、以下の用語を理解する

・リポジトリとは、ファイルやディレクトリを入れて保存しておく貯蔵庫のようなもの

・リモートリポジトリとは、特定のサーバー上に設置して複数人で共有するためのリポジトリのこと

・ローカルリポジトリとは、ユーザーごとに配置される手元のマシンで編集できるリポジトリのこと

・ブランチとは、ファイルの編集履歴を分岐させて記録していく機能のこと

・リモートブランチとは、リモートリポジトリ上のブランチのこと

・ローカルブランチとは、ローカルリポジトリ上のブランチのこと

・デフォルトブランチとは、git cloneをしたときにcheckoutされるブランチのこと。今回のbootcampでは、それがmainにあたる
  git remote show origin | grep 'HEAD branch' | awk '{print $NF}'というコマンドを使って、デフォルトブランチを取得することができる。

・ワークツリーとは、Gitで管理されている、実際に作業を行うディレクトリのこと

・インデックス・ステージとは、リポジトリに保存されている情報とワークツリーとの差を記録する場所。
  Gitにおいて、commitをするときは、ワークツリーから直接リポジトリ内に状態を記録するのでなく、
  その間に設けられているインデックスの設定された状態を記録するようになっている。
